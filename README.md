# detect-planes-trains-automobiles

A generic Python application to detect planes, trains, and automobiles.

## Requirements

- Python 3.11+

## Installation

```bash
pip install -e .
```

## Usage

Run via the module:

```bash
python -m detect_planes_trains_automobiles
```

Or use the installed script:

```bash
detect-planes-trains-automobiles
```

## Development

Install dependencies and run tests:

```bash
pip install -e .
python -m pytest
```
